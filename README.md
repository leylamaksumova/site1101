# Your Name — Portfolio

This repository contains a personal portfolio site built for the SITE1101 course using plain HTML, CSS, and JavaScript.

How to view locally
- Open `index.html` in a browser (double-click or right-click → Open with...). No build step required.

Publishing
- This site is intended to be published with GitHub Pages (branch `main`, root). After pushing to GitHub, enable Pages in the repository Settings.

Notes
- Replace placeholder content (your name, email, images, and GitHub links) before sharing.
- Project files live in the repository root: `index.html`, `about.html`, `projects.html`, `styles.css`, `script.js`, and `assets/`.
